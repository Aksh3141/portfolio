Multi-Agent RL for Token Pruning in Vision Transformers

Designed a multi-agent reinforcement learning framework using PPO to selectively prune tokens in Vision Transformers (ViTs), reducing computation while preserving accuracy. Key aspects:

  • Token importance learned via policy networks per attention head
  • Dynamic pruning strategies based on per-image complexity
  • Benchmarked with standard ViT models for performance-efficiency trade-offs

Gravitational Wave Glitch Detection

Developed machine learning models to detect and classify glitches in LIGO strain sensor data. Highlights include:

  • Preprocessing of time-series signals into spectrograms and statistical features
  • Implemented and compared SVM and MLP classifiers
  • Improved model interpretability using saliency maps and confusion matrix analysis
